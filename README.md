# Dashboard ENEM 2024 📊

Este projeto é um dashboard interativo feito em Streamlit para análise dos resultados do ENEM 2024.

## Autor
**Igor Andrade** – Disciplina de Análise Exploratória de Dados

## Objetivo
Visualizar e explorar os dados do ENEM 2024, permitindo filtrar por estado, cidade, faixa etária e outras variáveis importantes.
